FredMDQD.fred_transform — Method

Apply a FRED tranformation code

Arguments

  • tcode: FRED transformation code
  • x: a vector of data

Transformation Codes

The following transformation codes are currently supported

  1. $x$
  2. $\Delta x$
  3. $\Delta^2 x$
  4. $log(x)$
  5. $\Delta log(x)$
  6. $\Delta^2 log(x)$
  7. ``Δ(x{t+1} / xt - 1)
